## 현식의 코드
```sql
SELECT 
     ins.name
    , ins.datetime 
    from animal_ins ins 
    left outer join animal_outs outs
    on ins.animal_id = outs.animal_id
    where outs.animal_id is null
    and rownum < 4
    order by datetime ;
```
