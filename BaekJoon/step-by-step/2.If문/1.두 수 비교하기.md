- 두 정수 A와 B가 주어졌을 때, A와 B를 비교하는 프로그램을 작성하시오.  
  
## 현식의 코드

```java
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int A = sc.nextInt();
        int B = sc.nextInt();

        sc.close();
        if (A > B){
            System.out.println(">");
        }else if(A < B){
            System.out.println("<");
        }else{
            System.out.println("==");
        }

    }
}
```

## 현영의 코드

```java
import java.util.Scanner;

public class Main {

  public static void main(String[] args) {

    Scanner sc = new Scanner(System.in); 
    while(true) {
      int a = sc.nextInt();
      if (a < -10000 || a > 10000) {
        continue;
      }

      int b = sc.nextInt();
      if (b < -10000 || b > 10000) {
        continue;
      }

      if(a > b) {
        System.out.println(">");
        break;
      } else if(a == b) {
        System.out.println("==");
        break;
      } else {
        System.out.println("<");
        break;
      }
    }
    sc.close();
  }
```
